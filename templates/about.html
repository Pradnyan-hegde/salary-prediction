About SalaryPredict

SalaryPredict is an intelligent web-based application designed to estimate an individual's salary based on features like education level, years of experience, job role, and more.

This project demonstrates the practical application of machine learning in workforce analytics and is intended for educational and exploratory use only.

Features:

  • User-friendly interface for data input
  • Accurate salary prediction using a trained ML model
  • Clean and responsive design

Technologies Used:

  • Python & Flask (Backend)
  • Scikit-learn (ML Model)
  • HTML, CSS, JavaScript (Frontend)
